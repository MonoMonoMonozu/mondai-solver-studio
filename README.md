# 📋 課題解決ナビゲーター (Problem Solving Navigator)
**QuestQueries #023**

「モヤモヤを結晶化したらアプリになった」—— 複雑な課題をマンダラチャートとAIの力で紐解き、具体的なアプリ要件まで一気通貫で導き出す思考支援ツールです。

![App Screenshot](images/screenshot.png)

## 🌟 特徴
単なるタスク管理ツールではありません。ひらめきを呼び込むための「問い」を立てることを目的としています。

*   **マンダラ・ブレスト (PH.0):** 独自の8軸で思考を強制展開。
*   **AI仮説生成 (PH.2):** ヒアリング内容から根本原因の仮説と打ち手の方向性を提示。
*   **打ち手マトリックス (PH.3):** 効果と実行しやすさで優先順位を可視化。
*   **要件定義エクスポート:** そのまま開発に回せるMarkdown形式の仕様書を出力。

## 🚀 使い方
1.  このリポジトリの `index.html` をダウンロードします。
2.  ブラウザで `index.html` を開きます。
3.  画面右上の設定から Anthropic の API KEY を入力してください。
4.  Phase 0 から順に、AIとの対話を進めてください。
